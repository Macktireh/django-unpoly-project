# Django + Unpoly = ❤️🚀

### Installation

```bash
git clone https://github.com/Macktireh/django-unpoly-project.git
```

```bash
cd django-unpoly-project
```

```bash
pdm install
```

### Migrate Database

```bash
pdm run migrate
```

### Load Data

```bash
pdm run loaddata issues.json
```

### Run Server

```bash
pdm run start
```
